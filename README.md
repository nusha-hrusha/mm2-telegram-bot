# 🔪 Roblox MM2 Price Bot

Умный Telegram-бот для отслеживания цен на скины в игре Murder Mystery 2.
Бот автоматически собирает данные с сайта MM2 Values и предоставляет их пользователям по запросу.

## 🚀 Функции
*   **Скрепинг данных:** Автоматически обновляет цены раз в сутки.
*   **База данных:** Хранит данные в локальном JSON-файле (минимум нагрузки на сервер).
*   **Анти-спам:** Защита от частого использования команд в групповых чатах.
*   **Docker-ready:** Полностью настроен для быстрого деплоя.

## 🛠 Технологии
*   **Node.js** — ядро бота.
*   **Telegraf** — библиотека для работы с Telegram API.
*   **Axios & Cheerio** — инструменты для сбора данных с веб-сайтов.
*   **Node-cron** — планировщик задач для автообновления цен.

## 📦 Быстрый запуск (Docker)

1. Клонируйте репозиторий:
   ```bash
   git clone https://github.com/nusha-hrusha/mm2-telegram-bot.git
   cd mm2-telegram-bot
   ```

2. Создайте файл для данных (чтобы Docker правильно его подхватил):

	```bash
	touch items.json
	```

3. Создайте в корне проекта файл `.env` и поместите туда токен вашего Telegram бота (получить его можно через @BotFather)

	```
	TOKEN_BOT=ваш_токен_тут
	```

4. Запустите проект через Docker Compose:

	```bash
	docker-compose up -d --build
	```

## 🎮 Как пользоваться в Telegram
Добавьте бота в чат и используйте команду:
` /price [название оружия] `

Пример: `/price batwing`
